Featured Project — Threat Alert Bot (Demo)

Endpoint Threat Alert & Correlation

A compact automation that collects EDR-like telemetry, checks IP reputation, and pushes summaries to Slack/Webhook. Built to be readable, repeatable and safe to run in lab environments.

What it does
  • Parses endpoint logs for suspicious commands and failed authentications.
  • Checks external IP reputation via public API (demo mode).
  • Posts a short alert with evidence and suggested actions to a webhook.

How it works (behind the scenes)
1) A small Python script (threat_alert_demo.py) simulates telemetry and writes docs/alerts.json
2) A GitHub Actions workflow runs the script on push or schedule and commits alerts.json
3) GitHub Pages serves the site; the page fetches alerts.json and displays the latest alert
4) You can trigger the workflow manually from Actions or push changes to main
Why this matters

This automation reduces time-to-detect for endpoint compromises by correlating process telemetry with network reputation and surfacing only high-confidence alerts to analysts — cutting noise and enabling faster triage.
